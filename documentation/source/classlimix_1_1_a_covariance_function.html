Detailed Description

Abstract base class for covariance functions.

Covariance functions generate kernel matrices from inputs (

See Also
X) and a set of hyperparameters (
params). This abstract base class provides the interface and standard implementation for convenience function.

const muint_t  d 
) const

calculate the gradient of the covariance with respect to all entries X[:,d]

Parameters
outpoints to the target matrix
ddenotes the column of X

Note: formally, the gradients of K w.r.t any hyparparameter is again an NxN matrix (see for example ). However, if the gradient is w.r.t to a particular input X_{n,d}, only the nth row in that gradient matrix is non-zero. To safe computation, the function calculates the gradient with respect to all X[:,d] simultaneously where the entries reflect the non-zero row in each.
