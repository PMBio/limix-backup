Detailed Description

template<int rows, int cols, int options>
class limix::FlexEigenMatrix< rows, cols, options >

Wrapper to handle Eigen arrays of flexible type.

This is particularly used for row and header handling, which have unknown data types, See CHeaderMap for details.
