What is LIMIX?

LIMIX is a flexible and efficient linear mixed model library with interfaces to Python and other scripting languages. Currently, LIMIX is mainly developed by Christoph Lippert, Francesco Paolo Casale and Oliver Stegle.

Philosophy

Genomic analyses require flexible models that can be adapted to the needs of the user. LIMIX is smart about how particular models are fit to safe computational cost.

How to Use LIMIX?

A good starting point is our package Vignettes. These tutorials are available in a separate git repository: Limix-tutorials.

Problems and Documentation

If you want to use LIMIX and encounter any issues, please contact us by email: Limix team. For developers, you may find the automatic Doxygen source code documentation helpful.
